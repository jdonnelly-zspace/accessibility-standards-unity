KEY ACCESSIBILITY FEATURES BY VERSION
======================================

Unity 2021.3 & 2022.3:
- Limited dedicated accessibility features
- Basic editor accessibility (Scene view navigation)

Unity 2023.2:
- Introduced Accessibility Module
- Accessibility Hierarchy Viewer window (Window > Accessibility > Accessibility Hierarchy Viewer)
- Mobile screen reader API support (Android TalkBack, iOS VoiceOver)

Unity 6.0+:
- Enhanced Accessibility Module (enabled by default)
- AccessibilityNode for screen reader navigation
- AssistiveSupport for active hierarchy management
- AccessibilitySettings for platform accessibility settings
- VisionUtility for color-blind safe palettes
- Real-time Accessibility Hierarchy Viewer during Play mode
- Full mobile screen reader support (TalkBack/VoiceOver)

App UI Package:
- Keyboard navigation (Tab/Shift+Tab)
- Focus ring and visual focus indicators
- Layout direction support (RTL/LTR)
- KeyboardFocusController for managing focus behavior
